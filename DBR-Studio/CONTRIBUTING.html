How to Contribute

  1. Add a post: Edit posts/posts.json and commit. Each entry can have title, date, summary, tags, and optional image, embed, link, caption.
  2. Submit content via Issues: Use GitHub issue templates for Music/Art/Photography. Maintainers transfer highlights into posts.json.
  3. Polls: For persistent polls, create a GitHub Discussion or Google Form and link it from a post.

No build step needed. Everything is static.
